# Emoji 查找

> Emoji 搜索与选取工具，分类浏览关键词搜索，一键复制 Unicode 码点显示

## 功能概览

- **分类浏览：笑脸/人物/动物/食物/旅行/符号等分类筛选项**
- **关键词搜索：输入中文/英文关键词精准定位 Emoji**
- **点击复制 Emoji 字符，直接粘贴到任意应用**
- **显示 Unicode 码点（U+XXXXXX 格式）方便开发者引用**
- **常用 Emoji 收藏夹：高频使用的 Emoji 一键访问**

## 操作说明

1. 打开 index.html（浏览器直接运行，无需服务器）
2. 按界面引导输入/操作
3. 结果实时预览，支持一键复制导出

## 技术栈

- **HTML5**：语义化结构
- **CSS3**：白色极简主题，Flex/Grid 布局
- **Vanilla JavaScript**：纯原生代码，零依赖
- **ES6+**：sync/await、箭头函数、模板字符串
- **localStorage**：偏好设置持久化
